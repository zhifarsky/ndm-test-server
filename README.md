# Сборка
Собирается в DEB-пакет через Makefile
```
make
```
Также можно скачать DEB-пакет через [релизы](https://github.com/zhifarsky/ndm-test-server/releases).
# Установка
```
sudo apt install ./ndm-test-server_1.0_amd64.deb
```
После установки автоматически запускается systemd-сервис `ndm-test-server.service`

# Конфигурация
Файл конфигурации расположен в `/etc/ndm-test-server/config.ini`

Так он выглядит по умолчанию:
```
protocol=tcp
port=8080
```
- `protocol` можт принимать значения `tcp` или `udp`;
- после изменения конфигурации, можно отправить на сервер команду `/shutdown`. Сервер перезапустится с новыми настройками.

# Команды
Сервер принимает команды без символа перевода строки. Поддерживаются следующие команды:
- `/time` — возврат текущего времени и даты в формате "2025-11-10 17:28:45";
- `/stats` — возврат статистики (общее количество подключившихся клиентов и подключенных в данный момент);
- `/shutdown` — завершение работы.

# Пример использования
TCP

```
echo -n "/time" | nc localhost 8080
```
UDP

```
echo -n "/time" | -u nc localhost 8080
```
